# Peripheral Attention Trap

## Discovery Statement

In anti-money laundering graph monitoring, attention mechanisms systematically underweight peripheral nodes—entities with low direct transaction volume but high structural connectivity to core laundering clusters. The underweighting creates a blind zone where illicit funds accumulate through sink convergence: multiple low-signal edges route value toward a small set of peripheral hub nodes that eventually feed into high-confidence laundering paths. Detection models trained on historical alert data reinforce the blind zone because peripheral nodes rarely appear in labeled positive examples, producing a self-reinforcing feedback loop where the monitoring system learns to ignore precisely the structural features indicating emerging laundering topology.

## Intuition

Standard graph attention layers assign weight proportional to edge frequency and node degree centrality. Peripheral nodes—shell companies, dormant accounts, recently registered entities—exhibit low degree centrality at any single snapshot but maintain latent connectivity through temporal edges activating during laundering campaigns. The attention trap arises because the model's inductive bias favors high-activity nodes, causing peripheral sink convergence patterns to remain invisible until the laundering topology matures and the peripheral nodes transition into core nodes with high cumulative flow. By the time detection triggers, significant value has already moved through the network.
