# Peripheral Attention Trap

## Discovery Statement

In AML transaction monitoring, alert triage systems and human analysts allocate disproportionate cognitive and computational resources to high-degree hub nodes within a financial network, while peripheral nodes with low local connectivity accumulate undetected suspicious activity. Over time, illicit actors exploit the peripheral attention gap by routing funds through long chains of low-degree intermediaries that converge on a single sink entity. The convergence pattern remains invisible to hub-centric detection heuristics until the sink node triggers a threshold alert, at which point the peripheral chain has already completed multiple laundering cycles.

## Intuition

Standard graph-based AML monitoring applies PageRank-like attention scoring, which naturally amplifies visibility for nodes with high betweenness centrality. Adversarial actors who understand the attention allocation mechanism can construct transaction topologies where every individual edge falls below the alert threshold, yet the aggregate flow through a peripheral-to-sink funnel exceeds launderable volumes. The "trap" emerges when the monitoring system's own attention heuristic creates a blind zone: peripheral nodes receive near-zero inspection weight, and sink convergence only surfaces after temporal aggregation windows close. The pattern is isomorphic to the "long tail" exploitation seen in SEO spam networks, where low-authority pages collectively funnel link equity to a target.
