This appendix provides a complete SHACL shape graph for validating ADL Lite Level~3 (L3) relation blocks. The shape ensures that every relation conforms to the structural and semantic constraints of the ADL Core ontology: valid concept identifiers, closed predicate sets, confidence bounds, and mandatory directionality.

The SHACL shape is defined in \texttt{adl\_lite/shacl\_validation.py} and executed by \texttt{bash scripts/reproduce/reproduce\_shacl.sh}, which validates all exported PROV-O Turtle files against the built-in shape graph using \texttt{pyshacl}.
