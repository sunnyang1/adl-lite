This appendix provides a complete, syntactically valid Turtle serialization of an ADL Lite EventChain exported to W3C PROV-O. The example uses the \texttt{disc-capital-trap} concept, which undergoes registration, validation, challenge, and fork. The serialization demonstrates how each ADL event maps to a \texttt{prov:Activity}, how agent roles map to \texttt{prov:Agent}, and how cryptographic hashes map to PROV-O's provenance relations.

The PROV-O export is auto-generated by \texttt{adl\_lite/prov\_export.py}; running \texttt{bash scripts/reproduce/reproduce\_prov.sh} regenerates Turtle files for all example concepts and validates them with \texttt{rdflib}.
