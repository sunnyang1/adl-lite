## Event-first ontology refactoring (哲学重构)

Philosophical foundation: Wittgenstein Tractatus §1.1 — "The world is the totality of facts, not of things."

### Architecture shift
- **Before**: ADLFrontMatter mutates (status/confidence fields change in place). Concept = static object.
- **After**: EventChain append-only. Status/confidence/validators are COMPUTED from chain. Concept = event chain.

### New models
- `EventType` (enum): REGISTER, VALIDATE, DEPRECATE, FORK, ARCHIVE, RELATE, EVIDENCE, SEAL, ANNOUNCE, PUBLISH, SYNC_DASHBOARD, LISTEN, SNAPSHOT
- `Event` (Pydantic): event_id, concept_id, event_type, actor, reasoning, timestamp, payload, previous_event_id, hash
- `EventChain`: append-only, cryptographically linked. `.status`/`.confidence`/`.validators` computed from events. `.snapshot()` derives ADLFrontMatter. `.from_parsed()` factory. `.verify_integrity()`, `.history()`

### Model changes
- `ADLFrontMatter`: added `from_chain()`, `identity_dict()`; docstring now says "derived snapshot, not source of truth"
- `ADLDocument`: added `event_chain` property, `refresh_snapshot()` method

### Results
- 111/111 existing tests pass
- 7 event-first smoke tests pass (Event creation, chain integrity, status computation, from_parsed, refresh_snapshot, history)
